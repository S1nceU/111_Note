## Basic Text
#### Regular expressions
不管他是否開頭大小寫、複數都為同一個字

#### Disjunctions
+ **[wW]** 為裡面的其中一個字
+ **[A-Z]** 為範圍裡面的字
+ **^** 反向，在 **[]** 中為首位才表示否定
+ **|** 或
+ **?** 前一個字母可加可不加
+ **\*** 前一個字重複，0 ~ 無限
+ **+** 前一個字重複，1 ~ 無限
+ **.** 可代表所有字元
+ **^** 開頭
+ **$** 結尾

+ 可以用暫存器，用 () 當成暫存器，若不要可以用 (?:)，暫存器的名稱 <\\n>
