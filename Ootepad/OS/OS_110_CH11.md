## File System Implementation

#### On-Disk Structure
+ 必須要存在 Disk 上面，已經在硬碟上面了

+ Boot control block
	+ 通常會是第一個 partition
	+ UFS : boot block, NTFS : partition boot sector
+ Partition control block 
	+ partition 的細節
	+ UFS : superblock, NTFS : Master File Table
+ File control block 
	+ 檔案資訊
	+ UFS : inode, NTFS : stored in MFT ( relational database )
+ Directory structure
	+ 檔案組織
